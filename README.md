# 🧙‍♀️ The Witch

[🇷🇺 Русский](README.ru.md)

---

**The Witch** is a tiny platformer game created for **Nokia 3310 Jam #3**.  
### 🔗 [Build / Itch.io / WIN](https://zaza.itch.io/the-witch)

### 🎮 Controls (Keyboard)

- **Arrows** — Movement  
- **X** — Jump  
- **C** — Attack  
- **Esc** — Pause / Close dialog
