# 🧙‍♀️ The Witch

[🇬🇧 English](README.md)

**The Witch** — это небольшой платформер, созданный для **Nokia 3310 Jam #3**.  
### 🔗 [Билд / Itch.io / WIN](https://zaza.itch.io/the-witch)

### 🎮 Управление (Клавиатура)

- **Стрелки** — Движение  
- **X** — Прыжок  
- **C** — Атака  
- **Esc** — Пауза / Закрытие диалога
